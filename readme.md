# Configuration

Set-up and Configuration guideline

## Keys mapping

- `nt` - new tab
- `et` - edit tab

### bufferline
`Tab` - Next tab
`Shift + Tab` - Previous tab

### cmp
`Ctrl + b` - Scroll up
`Ctrl + f` - Scroll down
`Ctrl + Space` - Complete
`Ctrl + e` - Abort
`Enter` - Confirm

### git
`\gb` - git blame
`\go` - git browse

### telescope   
`q` - close
`N` - Create new file
`h` - go to parent dir
`/` - start insert
`/` - start insert
`;f` - find file
`;r` - live grep
`\\` - buffer
`;t` - help tags
`;;` - resume
`;e` - diagnostics
`sf` - file browse
